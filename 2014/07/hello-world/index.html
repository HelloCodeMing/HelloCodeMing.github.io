博客

花了几天时间，断断续续地改了一点东西，终于算是把博客搭好了。按照国际惯例，先写一篇hello world，简单地讲一下博客的搭建过程。

参照如何搭建一个独立博客-简明github pages与hexo教程，从域名申请，到github的配合，再到配置hexo主题，这篇教程各方面都写的很细致，基本上能解决搭建博客中的大部分问题了。

不过，在此过程中我还是遇到了一些坑，在这里提一下。

troubleshooting

  • git
    之前有使用git的习惯，以为这个静态博客也是只要用git工具传到github就可以了，但一直无法发布博客，会有submodule的错误，一直没有搞定。弄了半天才发现hexo本身就有提供deploy的工具，用hexo deploy就可以了，真是很囧。

  • deploy
    修改了本地的配置后，用hexo server工具在本地可以看到效果，但是deploy之后却无法在网站上看到效果，也纠结了很久，偶然见发现把generate的文件删除后重新generate可以解决这个问题。后来看到hexo的作者也提到了这个问题，

    他写的troubleshooting提到了很多问题，应该能有所帮助。

  • gitcafe
    开始是用github托管的，但是发现访问速度不是很理想，后来发现gitcafe也有同样的服务，转移之后发现效果很不错。
    另外有一个会影响访问速度的地方，就是google提供cdn服务，目前来说如果用了google或者其他cdn的css，js，网站打开的速度会受到严重影响，所以还是建议使用本地的css，js。

尾声

修改了favicon，配色，插件之后，也不太想去折腾博客了，毕竟博客还是以内容为主的。以后的一些心得，经验都会在博客上分享，同时也希望借助这个平台和更多的人交流。

这几天smartisan也陆续发货了，反响很好，看到支持了这么久的人获得成功自己也很是欣慰。最后也希望，认真的人都能成功。
